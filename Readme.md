## 🚀 Featured Projects

### 1) Agentic RAG — Multi-PDF Chatbot
A production-style multi-PDF RAG platform with authentication, persistent storage, feedback-driven improvement, hybrid retrieval, and deployment-ready architecture.

**Tech:** Python, Streamlit, LangChain, Gemini API, ChromaDB, SQLite, Docker, Azure, Tavily

**Highlights**
- Multi-PDF ingestion with token-aware chunking
- Persistent user login and per-user document storage
- SQLite-backed chat history, uploads, and feedback
- ChromaDB retrieval with configurable top-k search
- Streaming LLM responses with provenance-aware output
- Feedback loop for incremental reranker training
- Hybrid reranking for table/image-heavy PDFs
- Docker and Azure deployment support

[Repository](https://github.com/ayushaks999/RaG_Chatbot)

- ML-based prediction pipeline with explainable output
- Interactive dashboards for data exploration
